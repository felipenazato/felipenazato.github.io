3 Usando apenas as propriedades dos logaritmos, calcule

3.1 Respostas da questão 3

\[log_{10}6 = log_{10}2 + log_{10}3 = 0,77\]

\[log_{10}1,5 = log_{10}\frac{3}{2} = log_{10}3 - log_{10}2 = 0,17\]

\[log_{10}5 = log_{10}\frac{10}{2} = log_{10}10 - log_{10}2 = 1 - log_{10}2 = 0,70\]

\[log_{10}72 = log_{10}{2^3}{3^2} = log_{10}2^3 + log_{10}3^2 = 3log_{10}2 +2log_{10}3 = 1,84\]
